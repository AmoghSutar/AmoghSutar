# Hi, I'm Amogh Sutar 👋

### Robotics & Automation Engineer | From Factory Floors to Autonomous Systems

I build systems where **software logic meets physical constraints** — from discrete event simulation of million-sqft warehouses to perception stacks for mobile robots. My core belief: the best automation engineers understand the *entire* pipeline, from the CAD file to the control loop to the fleet-level optimization layer.

- 🎓 M.Sc. **Automation & Robotics** @ TU Dortmund — automated driving, sensor modalities (LiDAR/radar), environment modeling, trajectory prediction, mobile robots
- 🏭 Former **Simulation Engineer** @ Dematic (KION Group) — discrete event simulation of intralogistics systems with **AutoMod**
- 🔩 Started in hardware: CAD design, PLC-controlled assembly lines, and embedded systems — I've debugged problems you can't fix with a `git revert`
- 🚀 Currently building toward **AI-driven warehouse logistics**: making simulation, prediction, and optimization accessible beyond the expert-tools niche
